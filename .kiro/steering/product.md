# Product Overview

Laravel FrankenPHP Service Manager is a system administration tool for managing Laravel FrankenPHP services on Linux systems with SystemD.

## Purpose

Simplifies management of staging and production Laravel FrankenPHP services by providing:

- Service status monitoring (real-time)
- Service control (start, stop, restart)
- Two interfaces: CLI and web-based (Streamlit)

## Target Users

- Developers managing Laravel applications
- System administrators
- DevOps engineers

## Key Features

- Real-time service status checking for staging and production environments
- Full service lifecycle management (start, stop, restart)
- CLI with colored output and tabular formatting
- Web interface with authentication (username/password + CAPTCHA)
- Visual dashboard with interactive tables and toast notifications
- Port scanning utility for monitoring open ports and processes

## Language

All user-facing content, documentation, and UI text is in Indonesian (Bahasa Indonesia).
